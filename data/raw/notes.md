MOT17 Raw Dataset

This folder is intended to store the raw MOT17 dataset used for training and evaluating the pedestrian detection model.

The dataset is not included in this repository due to its large size.

Download

You can download the dataset from Kaggle:

https://www.kaggle.com/datasets/wenhoujinjust/mot-17

Usage

Download the dataset from the link above.

Extract the dataset.

Place the extracted files inside this directory:

data/raw/

The project preprocessing scripts will read the dataset from this location and convert it to the format required for training.

Dataset Source

Dataset: MOT17 (Multi-Object Tracking Benchmark)

The dataset contains annotated pedestrian sequences commonly used for multi-object tracking research.
